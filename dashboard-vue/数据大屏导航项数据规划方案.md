# 高校数据大屏导航项数据规划方案

## 概述

本文档对高校数据大屏的八个导航项进行数据源规划，明确每个导航项可使用的数据文件及对应指标，为数据可视化开发提供依据。

***

## 一、科学研究

**数据文件：** 科技处、图书馆

| 文件  | 可用指标                                                                                            |
| --- | ----------------------------------------------------------------------------------------------- |
| 科技处 | 各级项目数（国家级/部级/省级/厅局级/校级）、各级平台数、纵向项目经费、省部级奖项（一/二/三等奖）、专著、各类知识产权（发明/实用新型/外观设计专利、动植物新品种、标准发布、软件著作权） |
| 图书馆 | 高水平论文发表情况（Nature/Science/Cell、SCI一\~四区、EI、CSCD、核心期刊、其他）                                         |

**核心展示：** 科研项目、科研经费、科研成果（论文+专利+获奖+专著）

***

## 二、学科建设

**数据文件：** 教务处、研究生院、校办

| 文件   | 可用指标                                                                       |
| ---- | -------------------------------------------------------------------------- |
| 教务处  | 省级特色重点学科、省级重点学科、一流建设学科、一流学科、教育部特色专业、省级特色专业、省级示范专业、省级精品课程、本科专业总数、当年新增/停招专业数 |
| 研究生院 | 博士后流动站、一级博点、一级硕点、二级硕点、专业硕点                                                 |
| 校办   | 学院数                                                                        |

**核心展示：** 学位点布局（硕博点）、重点学科、特色专业、学科平台

***

## 三、人事人才

**数据文件：** 人事处、教务处

| 文件  | 可用指标                                                                                 |
| --- | ------------------------------------------------------------------------------------ |
| 人事处 | 教师总数、专任/管理/教辅人员数、学历结构（博士/硕士/本科）、职称结构（正高/副高/中级/初级）、外聘教师数、年龄分布（35岁以下/36-45/46-55/56以上） |
| 教务处 | 国家名师奖、省级名师奖、黄大年式教师团队、省部级教学团队、国家级/省级课程思政教学团队                                          |

**核心展示：** 师资规模与结构、高层次人才与教学团队

***

## 四、党建思政

**数据文件：** 组织部、统战部、教务处、研究生院

| 文件   | 可用指标                                      |
| ---- | ----------------------------------------- |
| 组织部  | 党员数                                       |
| 统战部  | 各民主党派党员数（民革/民盟/民建/民进/农工党/致公党/九三学社/台盟）     |
| 教务处  | 本科生中共党员、共青团员、民主党派、少数民族、残疾人、国家级/省级课程思政教学团队 |
| 研究生院 | 研究生中共党员、共青团员、民主党派、少数民族、残疾人                |

**核心展示：** 党员队伍规模、民主党派建设、学生政治面貌构成、思政教学团队

***

## 五、人才培养

**数据文件：** 教务处、研究生院、国际教育学院

| 文件     | 可用指标                                                                                          |
| ------ | --------------------------------------------------------------------------------------------- |
| 教务处    | 预科生/本科生数（分男女）、毕业人数/毕业率、学位授予人数/授予率、就业人数/就业率、开设课程总门数、教授讲授本科课程门次数、国家级/省级/校级教改工程项目、省级教学成果奖、校外实践基地 |
| 研究生院   | 硕士/博士生数（分男女）、导师数（分男女/硕博）、毕业人数/毕业率、学位授予率、就业人数/就业率                                              |
| 国际教育学院 | 中外合作办学招生数、留学生数                                                                                |

**核心展示：** 本硕博学生规模、毕业与学位授予、就业情况、教学改革、国际化办学

***

## 六、社会服务

**数据文件：** 科技处、教务处、研究生院、国际教育学院、人事处

| 文件     | 可用指标                                                                  |
| ------ | --------------------------------------------------------------------- |
| 科技处    | 横向项目经费数（校企合作）、发明专利（技术储备与转化）、实用新型专利（实用技术推广）、标准发布（行业标准服务）、软件著作权（技术成果输出） |
| 教务处    | 本科就业人数、本科就业率、校外实践基地（校企/校地合作平台）                                        |
| 研究生院   | 博士就业人数、博士就业率、硕士就业人数、硕士就业率                                             |
| 国际教育学院 | 留学生数（国际文化交流服务）、中外合作办学招生数（引进优质教育资源服务社会）                                |
| 人事处    | 外聘教师数（行业/企业专家参与教学）                                                    |

**核心展示：** 产学研合作与技术服务、人才输出服务社会、国际交流服务、行业专家参与

> 💡 **说明：** 社会服务可从四个维度展示：①产学研合作（横向经费、专利、标准、软著）体现技术输出；②就业数据（本硕博就业人数与就业率）体现人才服务社会；③国际教育（留学生、中外合作办学）体现国际交流服务；④外聘教师体现行业专家资源引入。

***

## 七、财政资产

**数据文件：** 国资处、科技处、图书馆

| 文件  | 可用指标                                                  |
| --- | ----------------------------------------------------- |
| 国资处 | 学校面积、教学行政用房/实验室用房/学生宿舍、固定资产、教学科研设备数及原值、50万元以上大型设备数及原值 |
| 科技处 | 纵向项目经费数（科研经费收入）                                       |
| 图书馆 | 图书数量、电子图书(种)、电子期刊(种)（文献信息资产）                          |

**核心展示：** 校园面积与基础设施、固定资产与仪器设备、科研经费收入、文献信息资源

***

## 八、校园安全

**数据文件：** 无直接对应

> ⚠️ **说明：** 现有10个数据文件中没有直接涉及校园安全的指标（如安全事故数、安全隐患排查、消防演练、安保人员等）。如需在此导航项展示数据，建议后续补充如：安全事故发生数、安全隐患整改率、消防演练次数、安保人员配备、监控系统覆盖数、心理健康干预人次等数据。
